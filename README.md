## Requirements
* [NodeJS](https://nodejs.org/en/)
* [Yarn](https://yarnpkg.com/lang/en/) (optional)

## Install

```bash
$ yarn install
$ yarn run dev
```

## Usage
Open [http://localhost:3000](http://localhost:3000)
